Cloudflare Workers AI Setup

Add your Cloudflare credentials in the API Keys tab first.

  1. Sign up at cloudflare.com (free account)
  2. Go to Workers & Pages to activate Workers AI free tier
  3. Note your Account ID (top-right of Workers overview page)
  4. Go to Profile → API Tokens → Create Token
  5. Use template "Edit Cloudflare Workers" — grants Workers Scripts: Edit + Workers AI: Execute
  6. Key format: ACCOUNT_ID|API_TOKEN
