Arbiter Gateway

An intelligent, OpenAI-compatible API gateway that routes requests across multiple free-tier LLM providers with automatic key rotation, rate-limit tracking, semantic caching, and zero vendor lock-in.

Base URL
https://arbiter.chkoushik.com
API Version
v1.0.0
Protocol
OpenAI-compatible REST
Quick Start curl 
curl -X POST https://arbiter.chkoushik.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer your-gateway-key" \
  -d '{
    "model": "gemini-2.5-flash-lite",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
Features
  • ✓OpenAI-compatible /v1/chat/completions endpoint — drop-in replacement
  • ✓8 providers: Gemini, Groq, Cerebras, Cloudflare, OpenRouter, Cohere, HuggingFace, Pollinations
  • ✓Intelligent routing based on model name, token count, and detected content type
  • ✓Automatic key rotation with weighted availability scoring across multiple accounts
  • ✓Response caching for deterministic requests (temperature ≤ 0.3) via Redis
  • ✓Vendor and model override via ?vendor= and ?force_model= query parameters
  • ✓Optional bearer token authentication via GATEWAY_API_KEYS
  • ✓Cloudflare Workers AI management API (create, list, delete workers)

Authentication

Bearer token auth for protected endpoints. Fully disabled when GATEWAY_API_KEYS is unset.

Bearer Token

Set GATEWAY_API_KEYS to one or more comma-separated keys. All protected endpoints require: 

Authorization: Bearer your-gateway-key

Multiple keys can be provided as a comma-separated list in GATEWAY_API_KEYS.

Endpoints

All endpoints follow OpenAI's REST conventions.

Chat Completions Protected
POST /v1/chat/completions

Create a chat completion. Accepts OpenAI-format request body and returns OpenAI-format response.

Query Parameters
vendor Force a specific provider: gemini, groq, cerebras, cloudflare, openrouter, cohere, huggingface, pollinations
force_model Override the model ID sent to the provider (e.g. llama3.1-8b). Overrides the request body model field.
Request Body
{
  "model": "gemini-2.5-flash-lite",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user",   "content": "Hello!"}
  ],
  "temperature": 0.7,
  "max_tokens": 512,
  "top_p": 1.0
}
Response
{
  "id": "chatcmpl-abc12345",
  "object": "chat.completion",
  "created": 1711234567,
  "model": "gemini-2.5-flash-lite",
  "choices": [{
    "index": 0,
    "message": {"role": "assistant", "content": "Hello! How can I help you today?"},
    "finish_reason": "stop"
  }],
  "usage": {"prompt_tokens": 24, "completion_tokens": 12, "total_tokens": 36}
}
List Models
GET /v1/models

List all available models across all configured providers. No authentication required.

{
  "object": "list",
  "data": [
    {"id": "gemini-2.5-flash-lite", "object": "model", "owned_by": "gemini"},
    {"id": "llama-3.1-8b-instant",  "object": "model", "owned_by": "groq"},
    ...
  ]
}

Cloudflare Workers AI Management
GET/cloudflare/models

List available Workers AI text-generation models.

POST/cloudflare/workers

Create a new AI Worker with a specified model binding.

{"name": "my-ai-worker", "model": "@cf/meta/llama-3.1-8b-instruct", "description": "My AI worker"}
GET/cloudflare/workers

List all existing workers in the account.

DELETE/cloudflare/workers/{script_name}

Delete a specific worker script by name.
