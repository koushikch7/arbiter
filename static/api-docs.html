What is Arbiter?

Arbiter is an intelligent LLM gateway that provides a single OpenAI-compatible endpoint aggregating multiple AI providers.


It automatically routes your requests to the best available provider based on model requirements, token count, and capability detection — with transparent fallback across providers and API keys.


Features: weighted key rotation, rate-limit tracking, semantic response caching, and real-time health monitoring.

Quick Start
Base URL
http://your-host:8000
Compatibility

Drop-in replacement for the OpenAI API. Set base_url to Arbiter's address in any OpenAI SDK.

from openai import OpenAI client = OpenAI( base_url="http://localhost:8000/v1", api_key="your-gateway-key" )
Routing Logic
Model Name Routing

If you specify a model like gemini-*, @cf/*, or command-r, it routes directly to that provider.

Token-Count Routing

Large context (>100K tokens) automatically routes to Gemini. Medium context (>16K) uses Gemini or OpenRouter.

Capability Routing

Code-related requests route to Gemini/Groq first. Small prompts (<4K tokens) prefer Groq for speed.

Authentication

Arbiter uses Bearer token authentication. Include your gateway API key in every request header:

Authorization: Bearer <your-gateway-api-key>

Configure gateway keys via the GATEWAY_API_KEYS environment variable (comma-separated). If unset, auth is disabled.

Exempt Paths (No Auth Required)
${['/health','/docs','/redoc','/openapi.json','/api-docs','/dashboard','/dashboard/stats','/v1/models','/settings','/settings/routing','/static/*'].map(p => `${p}` ).join('')}
Cloudflare Access (Optional)

Enable ENABLE_CF_ACCESS=true with CLOUDFLARE_ACCESS_TEAM_NAME for Cloudflare Zero Trust JWT validation via the Cf-Access-Jwt-Assertion header.

Chat
POST /v1/chat/completions OpenAI-compatible chat endpoint

Main chat completions endpoint. Supports all OpenAI SDK clients.

ParameterTypeRequiredDescription
modelstringrequiredModel identifier. Use auto for smart routing, or specify a provider model name.
messagesarrayrequiredArray of message objects with role and content.
temperaturefloatoptionalSampling temperature 0–2. Values ≤ 0.3 enable semantic caching.
max_tokensintegeroptionalMaximum tokens in the response.
streambooleanoptionalNot yet supported — pass false or omit.
vendorstringoptionalForce a specific provider: gemini, groq, cerebras, etc.
{ "id": "chatcmpl-abc123", "object": "chat.completion", "model": "gemini-3.1-flash-lite-preview", "choices": [{ "index": 0, "message": { "role": "assistant", "content": "..." }, "finish_reason": "stop" }], "usage": { "prompt_tokens": 12, "completion_tokens": 48, "total_tokens": 60 } }
Models
GET /v1/models List available models

Returns all models across configured providers in OpenAI-compatible format.

{ "object": "list", "data": [ { "id": "gemini-3.1-flash-lite-preview", "object": "model", "owned_by": "gemini" }, { "id": "llama-3.1-8b-instant", "object": "model", "owned_by": "groq" }, ... ] }
