Cache Management

The semantic cache stores responses for deterministic requests (temperature ≤ 0.3). Clearing the cache forces re-fetching from providers.

Warning

Clearing the cache will delete all cached responses. This cannot be undone. Subsequent identical requests will hit the LLM providers directly.
